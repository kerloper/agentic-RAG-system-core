# Agent Context

## Current Role
AI system designer and developer.

## Project Phase
Core pipeline complete.

## What Has Been Built So Far
- Express server with `/health` and `/api/ask`.
- Qdrant integration with auto collection setup.
- Embedding service via Ollama.
- Ingestion pipeline (read → chunk → embed → store).
- Retrieval, prompt builder, and LLM call.
- Knowledge base with markdown files in `/docs`.

## What Is NOT Built Yet
- Hybrid search.
- Reranking.
- Conversation memory.
- Streaming responses.
